https://raphaelcordonmovieapi.herokuapp.com/

The app connects to OMDb API in order to obtain a list of:
  - Movies
  - Series
  - Games
  - Live Concerts

OMDb API offers two types of search:

  - Multiple results:
I used this one to list all the results related to the name typed.
  - Specific data:
Once clicked on "More Details" from a movie, the detailed search according to the IMDB ID will be shown.
