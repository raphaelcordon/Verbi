http://resenhando.co (Portuguese only)

The app is a mini 'social-media' where the users can review an Artist, Albun or Music.
The data from artists and musics are brought from Spotipy, an API from Spotify, the user only needs to pick the item to be reviewed and write the review, pretty simple.


Since it's a side-project I gave myself the pleasure to see the final product done in my mother tongue. Also, since most of my friends are Brazilian, it made more sense to use Portuguese instead of English, though the code itself is all written in English (check on Github).
